IMPORTANT CHANGES:  Things you will need to do differently

To open a project:

-          if the Files pane is not already open, click on the Folder icon on the left margin to open it

-          In the Files Pane:

o    Double click on the name of the project you wish to open

o    Double click on the index.gui file to open the Design view for the project

 

To view properties & styles, select components and view the tree view:

The slide-out Palette pane has was on the right side of the Design view has been split into separate panes that are accessed by clicking on the following icons:

Properties

allows the properties of the selected component to be edited

Styles

allows the styles for the selected component to be edited

Component Palette

allows components to be dragged onto the Design

Project Outline (Tree View)

provides a project outline that shows the hierarchy of the components in the project, and allows non-visible components to be selected so that their properties and styles can be edited

Select Parent

click this icon (in the menubar) to select the parent of the currently selected component

Select Children

click this icon (in the menubar) to select the children of the currently selected component

 

Opening and Closing Files:

As before, to open a file, double click on it in the Files pane.

To close the file, hover the mouse over the Editor tab for the file.  Click on the icon that appears in the upper right corner of the tab to close the file.

 

Editing the gist (i.e. the HTML tags in your design)

-          As before, to open the gist editor, ensure the Design view for the project is visible and click the icon in the menubar

-          To go back to the design view for the project, click the icon in the menubar again. 

 

Multi-select:

-          Ctrl-click:  (shift-click no longer supported)
